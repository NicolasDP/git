pub struct Date(_);

Git date object

Based on the chrono crate.

It is always at the current Local time, without nanoseconds.

Example

Creating a new date for now:

use git::object::Date;

let now = Date::now();
println!("Today is: {}", now);Run

Creating a custom date:

use git::object::Date;

let date = Date::ymd_hms(2013, 1, 29, 15, 34, 11)
            .expect("to have a valid date and time");
println!("First steps in UK: {}", date);Run
